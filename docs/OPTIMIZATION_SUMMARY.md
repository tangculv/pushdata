## 🚀 使用新功能

### Webhook 配置

1. 点击"Webhook 配置"按钮（或按 `Ctrl+W`）
2. 在对话框中修改各数据源的 webhook URL
3. 可以调整批次大小和超时时间
4. 点击"保存"保存配置

### 快捷键

- `Ctrl+O`: 快速选择文件
- `Ctrl+S`: 快速保存配置
- `Ctrl+W`: 打开 Webhook 配置
- `F5`: 开始处理
- `F6`: 仅推送待上传
- `Esc`: 停止当前任务

### 日志功能

- **搜索**：在搜索框中输入关键词，实时搜索日志
- **导出**：点击"导出"按钮，将日志保存为文本文件
- **清除**：点击"清除"按钮，清空日志显示
- **右键菜单**：右键点击日志区域，可以复制、清除、导出

### 帮助和关于

- 点击"帮助"按钮查看使用帮助和快捷键说明
- 点击"关于"按钮查看版本和项目信息
